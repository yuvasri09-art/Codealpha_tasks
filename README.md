# CodeAlpha Internship – Internet of Things (IoT)

This repository contains the completed tasks for the **Internet of Things (IoT)** internship program by [CodeAlpha](https://www.codealpha.tech).

## Tasks Completed

### Task 1: Research Report – IoT in Real Life
A research report (800–1000 words) on **IoT Applications in Smart Homes**, covering the architecture, applications, benefits, challenges, and future trends of IoT-based smart home systems.

**File:** `Task1_Research_Report/IoT_Smart_Homes_Report.pdf`

### Task 2: Sensor-Based Simulation
A simulation of an LED ON/OFF system controlled by a sensor input, built using Proteus. Includes circuit screenshots and code explanation.

**Folder:** `Task2_Sensor_Simulation/`
- Simulation screenshots
- Code with explanation

### Task 3: IoT Prototype with Arduino/Raspberry Pi
A **Smart Home Automation** system using an ESP32 microcontroller, a clap/sound sensor, and a relay-controlled light, integrated with the Blynk IoT platform for remote app-based control.

**File:** `Task3_IoT_Prototype/Smart_Home_Automation.pdf`

Includes:
- Project working principle and architecture
- Components used (hardware & software)
- Circuit diagram
- Embedded C++ code (Arduino IDE)
- Output, results, and conclusion

## Tech Stack
- **Microcontroller:** ESP32
- **Sensors:** Clap/Sound sensor (LM393), IR/Light/Motion sensors (Task 2)
- **IoT Platform:** Blynk
- **Tools:** Arduino IDE, Tinkercad
