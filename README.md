# CodeAlpha Internship – Internet of Things (IoT)

This repository contains the completed tasks for the Internet of Things (IoT) internship program by CodeAlpha.

---

## Tasks Completed

### Task 1: Research Report – IoT in Real Life
A research report (800–1000 words) on IoT Applications in Smart Homes, covering the architecture, applications, benefits, challenges, and future trends of IoT-based smart home systems.

📄 **File:** `Codealpha_Task1_Research_Report.pdf`

---

### Task 2: Sensor-Based Simulation
A simulation of an Ultrasonic Parking Sensor system using Arduino Uno and HC-SR04, built in Proteus 8 Professional. Includes 4 LEDs, a buzzer, and Virtual Terminal serial output across 5 distance zones.

📁 **File:** `Codealpha_Task2_Sensor_Based_Simulation.pdf`

Includes:
- Proteus simulation screenshots
- Pin connections and circuit details
- Full Arduino code with explanation
- Virtual Terminal serial monitor output

---

### Task 3: IoT Prototype with Arduino/ESP32
A Smart Home Automation system using an ESP32 microcontroller, a clap/sound sensor, and a relay-controlled light, integrated with the Blynk IoT platform for remote app-based control.

📄 **File:** `Codealpha_Task3_Iot_Prototype.pdf`

Includes:
- Project working principle and architecture
- Components used (hardware & software)
- Circuit diagram
- Embedded C++ code (Arduino IDE)
- Output, results, and conclusion

---

## Tech Stack
- **Microcontrollers:** Arduino Uno R3, ESP32
- **Sensors:** HC-SR04 Ultrasonic Sensor, Clap/Sound Sensor (LM393)
- **IoT Platform:** Blynk
- **Simulation Tool:** Proteus 8 Professional
- **IDE:** Arduino IDE
